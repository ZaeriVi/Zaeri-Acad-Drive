## ✨ Features

- **Collapsible subgroups** — EEE Fundamentals and Math & Physics are broken into numbered subgroups (11x, 12x, etc.) that can be expanded or collapsed to reduce scrolling
- **Live status badges** — green "link uploaded" vs gray "link pending" so you always know what's available at a glance
- **Sticky nav bar** — jump between sections without scrolling back to the top
- **Scroll-aware tabs** — the active tab highlights automatically as you scroll through the page
- **Floating hero image** with layered shadows
- Fully responsive — works on mobile and desktop

## 🚀 Usage

This is a single static HTML file. No build tools, no dependencies, no setup.

1. Download or clone the repo
2. Drop your image assets (`pic-1.png`, `pic-3.jpg`, `icon.png`) in the same folder
3. Open `index.html` in any browser — or host it anywhere (GitHub Pages, Vercel, Netlify, etc.)

To add or update a Drive link, find the relevant `course-card` in the HTML and replace `YOUR_LINK_HERE` with the actual Google Drive URL. Change `class="course-card a-xxx pending"` to `class="course-card a-xxx uploaded"` and update the badge text to `link uploaded`.
